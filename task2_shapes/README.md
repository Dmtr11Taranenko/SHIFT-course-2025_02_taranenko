# Консольное приложение для расчета и вывода параметров геометрических фигур.
На данный момент в реализации имеется три фигуры - круг, прямоугольник и треугольник

---

### Круг
**Ввод:** радиус

**Вывод:** название фигуры, площадь, периметр, радиус, диаметр

---

### Прямоугольник
**Ввод:** длины сторон (два значения)

**Вывод:** название фигуры, площадь, периметр, длина диагонали, длина, ширина

---

### Треугольник
**Ввод:** длины сторон (три значения)

**Вывод:** название фигуры, площадь, периметр, длина каждой стороны и противолежащий угол

---

# Запуск

#### Набор параметров:
 - **-i / --input** - параметр для установки пути вводного файла с входными данными для фигуры (Обязательный параметр)
 - **-o / --output** - параметр для установки пути файла для вывода (при отсутствующем значении вывод производится в консоль)
 - **-h / --help** - параметр для вывода сообщения с подсказкой

### Вариант 1)
**Через IDE**
 - В параметрах конфигурации запуска приложения ввести в поле _"Program arguments"_ строку входных аргументов:
   - -i {Ваш путь до input.txt} -o {Ваш путь до output.txt} -h (Для вывода сообщения с подсказкой)
 

 - Запустить приложение с помощью метода main

### Вариант 2)
**Через консоль**
 - Собрать Jar файл с зависимостями через консоль
   -  ./gradlew clean shadowJar


 - Находясь в папке репозитория ввести в консоль:
   - java -jar task2_shapes/build/libs/task2_shapes-1.0-SNAPSHOT-all.jar -i {Ваш путь до input.txt} -o {Ваш путь до output.txt} -h {Для вывода сообщения с подсказкой}
